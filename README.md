# 📊 КФИ — Кредитно-Финансовый Индекс

> Автоматизированная система анализа эмитентов высокодоходных облигаций (ВДО) по авторской методологии

## 🎯 О проекте

Система ежедневно анализирует эмитентов корпоративных облигаций на российском рынке, рассчитывает интегральный **Кредитно-Финансовый Индекс (КФИ)** по шкале 0–100 и публикует результаты в виде визуальных аналитических карточек.

### Что делает система:
- 📥 Автоматически собирает финансовую отчётность и данные о выпусках
- 🧮 Рассчитывает 30+ финансовых показателей по 6 аналитическим блокам
- 📊 Генерирует визуальную карточку анализа в стиле Apple Design
- 📱 Публикует результаты на сайте, в Telegram-канале и в Пульсе

## 📐 Методология КФИ v2.0

| Блок | Название | Вес |
|------|----------|-----|
| A | Платёжная устойчивость (ликвидность) | 20% |
| B | Структурная устойчивость (капитал и долг) | 25% |
| C | Операционная эффективность (рентабельность) | 10% |
| D | Денежный поток (реальность прибыли) | 25% |
| E | Облигационный профиль (специфика ВДО) | 15% |
| F | Качественная оценка (нефинансовые факторы) | 5% |

### Шкала КФИ:
| Диапазон | Категория | Цвет |
|----------|-----------|------|
| 80–100 | Надёжный | 🟦 Индиго |
| 60–79 | Стабильный | 🔵 Синий |
| 40–59 | Уязвимый | 🟡 Янтарный |
| 20–39 | Слабый | 🟠 Коралловый |
| 0–19 | Критический | 🔴 Красный |
