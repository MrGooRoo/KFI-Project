## Структура данных

### Формат хранения

```
data/financials/{emitter_id}_{period}.json
```

**Пример**: `CARMONEY_2024-Q4.json`

### Поля данных

**Баланс (9 полей)**:
- current_assets, inventory, short_term_investments
- cash, equity, long_term_debt
- current_liabilities, short_term_debt, total_assets

**ОПУ (4 поля)**:
- revenue, operating_profit, interest_expenses, net_income

**ОДДС (2 поля)**:
- cfo, capex

**Отраслевые (по группам)**:
- МФО: npl_90_plus, loan_loss_reserves, nmfk1_ratio, interest_income
- Лизинг: leasing_portfolio_net, lease_payments_received, overdue_portfolio
- Девелопмент: escrow_balance, project_finance_debt

## Проблемы и решения

### Проблема 1: e-disclosure.ru блокирует запросы

**Симптомы**:
- 403 Forbidden при requests
- Timeout при Playwright

**Причина**: Защита от ботов, медленная загрузка страниц

**Решение**:
- ✅ Создан ручной ввод с валидацией (работает)
- ⏳ Playwright скрапер (требует доработки)
- ⏳ Альтернатива: СПАРК API (платный)

**Текущее состояние**:
- 10 эмитентов в базе

**Решение**:
