## 🎯 Преимущества конвейера

1. **Автоматизация**: Один скрипт вместо 5 отдельных команд
2. **Интеллектуальность**: Автоматически определяет, нужен ли ручной ввод

5. **Удобство**: CLI для автоматизации, интерактивный режим для ручной работы

```bash

# 3. Ввести данные вручную (интерактивно)
# 4. Получить готовую карточку и тексты
```

### Сценарий 3: Пакетная обработка
```python
from pipeline import EmitterPipeline

pipeline = EmitterPipeline()
for emitter_id in ['CARMONEY', 'GTLK', 'SETL']:
    result = pipeline.run(emitter_id, skip_parsing=True)
    print(f"{emitter_id}: КФИ {result['kfi_result']['kfi_final']}")
```
