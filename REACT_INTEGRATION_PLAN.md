## Анализ исходного проекта

### Технологии
- **Frontend**: React 19.2.3 + TypeScript 5.9.3
- **Стили**: Tailwind CSS 4.1.17
- **Роутинг**: React Router DOM 7.14.2
- **Графики**: Recharts 3.8.1
- **Сборка**: Vite 7.2.4

### Структура
```
review-kfi-project-documentation/
├── src/
│   ├── App.tsx                 # Главный компонент с роутингом
│   ├── main.tsx                # Точка входа
│   ├── components/
│   │   ├── Header.tsx          # Шапка сайта
│   │   ├── KFIBadge.tsx        # Бейдж с КФИ
│   │   └── BlockBar.tsx        # Прогресс-бар блоков
│   ├── pages/
│   │   ├── RatingPage.tsx      # Главная страница с рейтингом
│   │   ├── EmitterPage.tsx     # Страница эмитента
│   │   └── MethodologyPage.tsx # Страница методологии
│   ├── data/
│   │   └── emitters.ts         # Данные эмитентов (6 примеров)
│   └── utils/
│       └── cn.ts               # Утилита для классов
├── package.json
├── tsconfig.json
├── vite.config.ts
└── index.html
```

### Ключевые компоненты

**RatingPage.tsx** (346 строк):
- Таблица и карточки эмитентов
- Фильтры по категориям и отраслям
- Поиск по названию/ИНН
- Сортировка
- Переключение вида (таблица/карточки)

**EmitterPage.tsx** (~400 строк):
- Детальная карточка эмитента
- 6 блоков анализа (A-F)
- Флаги рисков
- Информация об облигациях
- Графики и визуализация

**MethodologyPage.tsx** (~600 строк):
- Полное описание методологии КФИ v2.0
- Блоки A-F с формулами
- Отраслевые группы
- Шкала категорий
