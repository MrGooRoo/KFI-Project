# 📋 Готовые промпты для каждого чата

## Как использовать этот файл:
1. Скачайте его и загрузите в GitHub → `docs/prompts.md`
2. Перед каждым новым чатом откройте его на GitHub
3. Скопируйте нужный промпт
4. Вставьте в новый чат Arena.ai
5. Замените `ВАШЕ_ИМЯ` на ваш GitHub username
