3. **Создать страницу методологии**
   - Объяснить расчёт КФИ простым языком
   - Показать примеры для каждого блока
   - Добавить FAQ

4. **Настроить автоматизацию**
   - Еженедельное обновление данных MOEX
   - Автоматическая генерация карточек
   - Уведомления при изменении категории

### Для деплоя

1. **Выбрать хостинг**
   - Jino VPS (рекомендовано в roadmap)
   - Требуется Python + возможность запуска скриптов

2. **Настроить домен**
   - Купить домен (например, kfi-rating.ru)
   - Настроить DNS

3. **Деплой файлов**
   - Загрузить web/ на сервер
   - Настроить cron для обновления данных
   - Настроить SSL сертификат
