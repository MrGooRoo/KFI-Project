**Статус**: Требует доработки

**Описание**: Только 3 эмитента имеют реальные финансовые данные  
**Влияние**: Расчёты КФИ для 7 эмитентов на демо-данных  
**Приоритет**: Высокий  
**Статус**: Требуется сбор данных

**Влияние**: Визуальное, не влияет на функциональность
